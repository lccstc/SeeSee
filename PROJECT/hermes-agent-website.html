项目介绍

Hermes Agent 是 Nous Research 开源的 AI 智能助手框架。它不是简单的 ChatGPT 套壳——它有记忆、能执行命令、跨平台工作、支持技能扩展，真正成为你的数字伙伴。

Hermes 运行在你的终端里，同时可以通过 Telegram、WhatsApp、Discord、Slack、Signal 等 20+ 个消息平台与你对话。它能读写文件、执行命令、操控浏览器、搜索网页、定时执行任务，甚至用语音和你交流。

最重要的是——它记得你是谁。通过 MEMORY.md 和 USER.md，你的偏好、项目信息、工作习惯在每次会话之间持久保存。

使用场景

不只是聊天，是真正的生产力工具

🤖 个人 AI 助手

管理日程、查邮件、搜信息、写文档。在手机上通过 Telegram/WhatsApp 随时随地使用。它记得你的偏好，不需要每次重复说明。

👨‍💻 开发者工具

读代码、跑测试、调试 bug、管理 GitHub PR。在终端里直接对话式开发。搭配 Claude Code/Codex 做编码 agent 的调度中枢。

🔄 自动化工作流

定时监控网站变化、自动备份、数据采集、报告生成。用 cron 任务 + 技能包构建无人值守的自动化管道。

📱 多设备协作

终端、手机、平板——同一个 AI，在任何设备上继续对话。记忆和上下文通过文件同步，无缝切换。

快速上手

三步开始你的 AI 伙伴之旅

1

安装

pip install hermes-agent 或 brew install hermes-agent

2

配置 API Key

hermes setup 按提示配置模型提供商（OpenAI、Anthropic、本地模型等）

3

开始对话

hermes 进入交互模式，或 hermes "帮我查一下天气" 单次查询

+

接入消息平台

hermes gateway setup 配置 Telegram/WhatsApp 等，在手机上随时使用

如何选择？

选 Hermes Agent 如果你……

  • ✅ 偏好轻量、简洁的工具
  • ✅ 需要精确控制 token 成本
  • ✅ 想用本地模型降低成本
  • ✅ 主要是个人使用场景
  • ✅ Python 技术栈偏好

选 OpenClaw 如果你……

  • ✅ 需要覆盖更多平台（微信、钉钉）
  • ✅ 需要多设备深度协作
  • ✅ 喜欢丰富的社区插件市场
  • ✅ 团队使用场景
  • ✅ 需要可视化交互界面
