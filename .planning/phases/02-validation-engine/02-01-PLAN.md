<threat_model>
- If validator results are mixed into candidate rows, replay and comparison custody will blur immediately.
- If validation results are not durable, later publisher logic will be forced to recompute custody decisions ad hoc.
- If stable reason codes are missing, exception replay and industrialization will not have reliable machine-readable inputs.
</threat_model>

<verification>
This plan is incomplete unless PostgreSQL-backed persistence tests pass.
</verification>
